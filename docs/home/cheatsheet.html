Cheatsheet

Short instructin on how to create the .md files

Sections

Use five or more underscores ( _____ ) to separate sections.

Section processing

|[FUNCTION CALL]

Mark the first line of a section with a pipe-symbol ( | ) + the name of the processing function you want to apply to the section. Currently available:

  1. Container
  2. Collapsible
  3. Iframe
  4. Card
  5. Table
  6. Flextable
  7. Markdown

If a section has no function call, it will be rendered as markdown. In case you do not want to render a section at all, you can use |SKIP

Variables

|[FUNCTION CALL]:[variable]

Some processing functions take an extra variable. Add the variable after the |FUNCTION_CALL separated by a colon :

Container

|CONTAINER:[css-class]

The container function wraps the section in a div-container. Variable is required: refers to the class name of the wrapping container.

Example

|CONTAINER:page__synopsis
Sharki supermachine!

Output

Sharki supermachine!

Collapsible

|COLLAPSIBLE
[### [LABEL NAME][:checked]]
[COLLAPSIBLE CONTENT]

The collapsible function creates one or more collapsible containers. First call the function. Then in the following line state the label name: ### [LABEL NAME] Followed by the content of the container. If you want the container to be expanded by default add :checked to the label name. You can add multiple containers in a section.

Example

|COLLAPSIBLE
### Stevie Storm:checked
There come Stevie Storm and friends
They laugh and joke and demonstrate the disco-dance
He's got a lucky face
He gallops, to amaze
imaginary horse to race

Output

There come Stevie Storm and friends
They laugh and joke and demonstrate the disco-dance
He's got a lucky face
He gallops, to amaze
imaginary horse to race

Card

|CARD
[[key], [value]]

The card function creates a card from a comma separated table with two columns.

Example

|CARD
Name, Harry den Hartog
Pseudonym, Harry Merry
Born, 15 November 1971
Country, The Netherlands

Output

Name
Harry den Hartog
Pseudonym
Harry Merry
Born
15 November 1971
Country
The Netherlands

Table

|TABLE:[css-class]
[CSV]

The table function creates a html-table from a comma separated table.
Variable is optional: it adds a css class to the table.

Example

|TABLE
Album, Year, label
First Contact, 2007, Olé Records
Veronika, 2008, Tocado-Records
Australian Sun, 2012, Meeuw Muzak
Orama A-Delic Palooza O-Matic, 2014, Blowpipe

Output

Album Year label
First Contact 2007 Olé Records
Veronika 2008 Tocado-Records
Australian Sun 2012 Meeuw Muzak
Orama A-Delic Palooza O-Matic 2014 Blowpipe

Flextable

|FLEXTABLE
[CSV]

The flextable function creates a css table from a comma separated table. It has the property of realigning the rows into columns when viewed on a smaller viewport.

Example

|FLEXTABLE
First line, Second line, Third line, Fourth line
Animated whirlpool, Thermonuclear stool, Stays responsible school, 'Capricious, cool.'
Puzzling merry-go-round, 'Fascinating, profound', Enormous ego drowned, Me in show-sound.
Vibrant imagination, Distorts investigation, Into salacious, gratification.
Fertile fluorescence, Bioluminescence, Lavish phosphorescence, Opalescence.

Markdown

|MARKDOWN

The markdown function may be explicitly called. But any sections without a function call will be rendered as markdown anyway. For the rules of markdown refer to the Daring Fireball .
