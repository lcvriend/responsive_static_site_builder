Site structure table

The site structure table contains the structure for the site. It determines the hierarchy and order of the site pages.

A site that is built with this tool will have the following hierarchy:

Site

Sections

Chapters

Groups

Pages

This hierarchy is defined in the site structure table, which has the following columns:
