A tool for building responsive static sites for instruction manuals

The site builder for manuals was created for the student administration of Utrecht University . It is a tool that helps you create responsive static websites. The design is geared to use as instruction manual. It can be downloaded from GitHub .

Check out this website if you want to get an impression of what the tool can do. The cheatsheet gives an overview of the html elements you can use. The pages on the site structure table and the [html templates] explain how the builder sets up the site interface for you. If you know a bit about Jinja2 and css then it should not be too difficult to modify the layout.

Some features

  • Custom html elements (see the cheatsheet ).
  • Sitemap and easy site navigation.
  • Automatic cross-referencing.
  • Version builds.
  • Easily extendable and customizable.

How does it work?

There are three main components to the tool:

Component
Description
Component

Content

Description
The content folder acts as a simple CMS. This is where you edit the content of the site. The pages are stored as markdown text and the structure of the site in an excel table.
Component

Templates

Description
The layout of the website is stored in the templates folder. You can edit the .html and .css templates here.
Component

Scripts

Description

Two python scripts create the site:

  • The build site script builds the site from the content and templates
  • The build structure script is a tool for creating and updating the content folder.
Component

Modules

Description
The site builder modules contain the underlying code. The scripts need these to run.

About this tool

This tool constitutes my first (non-trivial) Python project. Creating this simple site builder has been a valuable learning experience for me. I have set up things as clearly and intelligently as I could. Nonetheless, there will undoubtedly be plenty of awkward and/or backwards code in the codebase. If anyone has feedback on this little project, I would be glad to hear it!
