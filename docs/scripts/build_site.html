Build site

This script will build the site in the output folder designated in config.ini

  • The script will render all .md files with a valid page id in the content folder and store them in the output folder.
  • It will copy the iframes and images folders to the output folder.
  • It will store properties.ini with the updated build version in the content folder.

Any time you run the build site script it will increment the build number (which is printed in the footer of the page). If you want to run this script without incrementing the build number, flag it with no_increment .
